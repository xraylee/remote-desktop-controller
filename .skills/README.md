# 项目 Skills

本项目使用的工作流 Skills 归档，确保团队成员和 AI Agent 能在项目上下文中直接引用。

## 目录结构

```
.skills/
├── product-management/          # 产品管理插件（8 个子技能）
│   ├── PRD生成/SKILL.md         # PRD 文档生成
│   ├── 产品脑暴/SKILL.md        # 结构化创意发散
│   ├── 产品指标复盘/SKILL.md     # 指标分析与归因
│   ├── 用户反馈分析/SKILL.md     # 反馈聚类与洞察
│   ├── 用户故事拆解/SKILL.md     # Epic → User Story
│   ├── 竞品分析/SKILL.md        # 功能对比与 SWOT
│   ├── 路线图更新/SKILL.md      # 迭代状态汇总
│   └── 需求优先级排序/SKILL.md   # RICE/ICE/MoSCoW
├── frontend-design/             # 前端视觉设计指导
│   └── SKILL.md
└── brainstorming/               # 架构/技术设计脑暴流程
    ├── SKILL.md
    ├── visual-companion.md      # 可视化辅助指南
    ├── spec-document-reviewer-prompt.md
    └── scripts/                 # 可视化伴侣服务脚本
```

## 在本项目中的使用记录

| Skill | 使用场景 |
|-------|---------|
| 产品脑暴 | 自研方案创意发散（V1/V2 脑暴文档） |
| PRD生成 | 生成 PRD v1.0（功能清单、交互流程、验收标准） |
| frontend-design | 产品原型设计（v1→v2→v3 迭代） |
| brainstorming | 技术架构设计（方案对比 → 逐节设计 → 规格文档） |
