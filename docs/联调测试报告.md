## 修复的问题

### 1. 黑屏问题
**原因：** `build_rust_lib.sh` 中 `BUILT_PRODUCTS_DIR` 未设置时，库文件路径错误
```
❌ Debug/Contents/Frameworks/librdcs_core.dylib
✅ Debug/rdcs_client.app/Contents/Frameworks/librdcs_core.dylib
```

**修复：** 完善路径判断逻辑

### 2. 服务器地址配置错误
**原因：** 默认配置使用了错误的端口
```
❌ ws://localhost:8080
✅ wss://localhost:8443
```

**修复：** 更新 `app_config.dart` 默认值
