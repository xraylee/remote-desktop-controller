## 📊 项目概览

RDCS (Remote Desktop Control System) 是一个面向小型团队（<50人）的远程桌面解决方案，采用 Rust + Flutter + Go 技术栈，支持 P2P 直连和中继回退的三层连接架构。

### 核心技术特性
- **三层连接架构**: L1 局域网 mDNS → L2 P2P NAT 穿透 → L3 中继回退
- **端到端加密**: Ed25519 密钥交换 + ChaCha20-Poly1305 加密
- **高性能编解码**: H.264 硬件加速，目标 1080p60 @ CPU <30%
- **完整的文件传输**: 分块传输 + 断点续传 + SHA-256 验证
